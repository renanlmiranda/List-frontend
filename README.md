#List - Frontend
 
Link: [Projeto](https://distracted-archimedes-ccf41a.netlify.app)

## Iniciar o projeto

- Clonar repositório: git clone {link}.
- Instalar dependências: yarn install.
- Criar .env baseado no .env.example
- Iniciar projeto: yarn start

## Dependências

- Reactstrap
- React-icons
- Styled-components
- Eslint/Prettier
- Axios
